How to Install

macOS

Method 1: Homebrew (Pro)

brew install --cask ilhan-mstf/tap/omnitool

Method 2: Manual Bypass

Download the .dmg, move to Applications, then Right-Click > Open to bypass the developer warning.

Method 3: Terminal

If blocked, run this to un-quarantine:

sudo xattr -rd com.apple.quarantine /Applications/Omnitool.app

Windows & Linux

Windows (.msi)

Run the installer. If prompted by SmartScreen, click "More info" and then "Run anyway".

Linux (.AppImage)

Make the AppImage executable before running:

chmod +x Omnitool.AppImage
